Sign up for an account

Simple web app exercise for account registration. Note that there is no security whatsoever. This should only ever be deployed locally and you should only use dummy data.

For more details, see https://github.com/skhzhang/account-signup-ex.

Account Information

Must be alphanumeric.
Must have a minimum length of 5 characters.

Must contain at least 1 number, 1 uppercase, and 1 lowercase letter.
Must have a minimum length of 8 characters.
